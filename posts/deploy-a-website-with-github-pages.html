How to Deploy a Website with GitHub Pages

GitHub is a great software development platform. Here, you will learn how to deploy a website with GitHub Pages, which is an easy way to deploy a website from a GitHub repository.

1. Sign Up for GitHub

If you already have a GitHub account, skip to Step 2.
Click here to sign up for GitHub.

You will see a sign-up form. Enter a username, email address and password. You can choose below if you want to receive product updates, announcements and offers through email. To verify your account, scroll down and click "Verify" and solve the puzzle.

Once you have filled everything in, click "Create account". Next, you will have to fill in some more details to personalize GitHub. Then, you will be asked to verify your email address. Go to your inbox and open the verification email. Click the verification link to verify your email address. Once all that is done, your account will be ready.

2. Creating a Repository

If you don't know anything about Git or GitHub, we recommend learning the basics first, especially if you want to use GitHub for more than just creating one website.
To create a new repository, click "New" on the left.

Now, make sure you name the repository exactly YourUsername.github.io and replace "YourUsername" with your actual username. Naming it right is important. If you name it wrong, your website won't be published!

After you're done, click "Create repository".

3. Upload/Create Files

Now once your repository is created, it's time to add files. if you already have the files for your website on your computer, click "Upload files". There, you can drag and drop files or open the file chooser. If you haven't created any files for the website, you can just click "Create new file". Name it "index.html", because that's the name web browsers expect for the main page of your website. Type your code, click "Commit changes" and your website will be published soon. To view your website, go to YourUsername.github.io (replacing "YourUsername" with your actual username) and use the shortcut Ctrl + Shift + R or Cmd + Shift + R until your website appears. This shortcut will clear the browser cache and reload the site to see it quickly (cache files are stored by the browser to speed up the loading time of a website). If that shortcut doesn't seem to do anything, try Ctrl + F5. Your website is now ready!
